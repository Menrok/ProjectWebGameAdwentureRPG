# Informacje ogólne o projekcie

## Dane podstawowe

Nazwa projektu: **Project Web Game Adventure RPG**  
Typ projektu: **Projekt indywidualny**  
Rodzaj gry: **Webowa gra przygodowa RPG (tekstowa / klikana)**  
Platforma: **Przeglądarka internetowa (SPA – Single Page Application)**

---

## Wykorzystywane technologie

- **Backend / logika gry:** C# – ASP.NET Core (Web API)
- **Frontend:** Vue 3 (TypeScript, Composition API, Vite)
- **Architektura:** Oddzielony frontend i backend
- **Baza danych:** SQLite
- **Komunikacja:** HTTP (REST, JSON)

---

## Opis koncepcji gry

Projekt zakłada stworzenie webowej gry przygodowej RPG, w której gracz eksploruje świat gry poprzez podejmowanie decyzji, wykonywanie akcji oraz stopniowy rozwój postaci.

Rozgrywka oparta jest na interakcjach typu „kliknij → odpowiedź systemu”, gdzie frontend pełni rolę interfejsu użytkownika, a cała logika gry (walidacja akcji, stan gry, konsekwencje decyzji) znajduje się po stronie backendu.

Gra będzie rozwijana iteracyjnie, zaczynając od trybu singleplayer oraz prostego głównego widoku gry dostępnego po zalogowaniu użytkownika.

Możliwość rozszerzenia projektu o wersję mobilną lub tryby wieloosobowe pozostaje otwarta, jednak nie jest celem początkowym projektu.
